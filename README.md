# FUTURE_PE_02 - AI Content Marketing Using UGC Ads

## Project Overview

This repository contains a complete AI-generated UGC Ad Content Pack for a realistic D2C skincare product. The project demonstrates how structured prompt engineering can generate conversion-focused short-form ad content for Instagram Reels, Facebook video ads, and YouTube Shorts.

## Task

Create an AI-generated UGC ad content pack including:

- Multiple scroll-stopping hooks
- UGC-style ad scripts
- Problem-solution-CTA ad formats
- Platform-specific content adaptations
- Captions and CTA options
- Prompt logic and reusable prompt templates

## Product Used

**GlowNest Barrier Repair Gel Cream**

A lightweight daily moisturizer for young adults with dry, sensitive, or irritated skin. The product positioning focuses on barrier repair, non-greasy hydration, and simple skincare routines.

## Prompt Engineering Approach

The prompt system uses four stages:

1. **Business and audience framing** - defines product, audience pain points, objections, and desired conversion action.
2. **Hook generation** - creates multiple first-three-second ad openings using curiosity, pain point, contrast, and social proof angles.
3. **Script generation** - turns selected hooks into UGC scripts with natural spoken language.
4. **Platform adaptation** - adjusts length, pacing, caption style, and CTA for Instagram, Facebook, and YouTube Shorts.

## Deliverables

- Prompt strategy and logic
