# Reusable Prompt Templates

## Prompt 1: Business Context Builder

```text
You are a senior UGC ad strategist and prompt engineer.

Create a concise marketing brief for the following product:

Product:
[PRODUCT NAME AND DESCRIPTION]

Target audience:
[TARGET AUDIENCE]

Main problem solved:
[CUSTOMER PROBLEM]

Brand voice:
[BRAND VOICE]

Conversion goal:
[CONVERSION GOAL]

Return:
1. Customer pain points
2. Customer objections
3. Emotional triggers
4. Product benefits
5. Content angles for UGC ads
Use practical language and avoid exaggerated claims.
```

## Prompt 2: Hook Generator

```text
Using the marketing brief below, generate 20 UGC ad hooks for short-form video.

Brief:
[PASTE MARKETING BRIEF]

Requirements:
- Write hooks for the first 3 seconds of a video.
- Include a mix of pain point, curiosity, mistake, contrast, routine, and social proof hooks.
- Keep each hook under 14 words.
- Make the hooks sound natural, not like polished brand copy.

Return the hooks in a table with:
Hook, Hook Type, Best Platform, Why It Works.
```

## Prompt 3: UGC Script Generator

```text
You are writing authentic UGC ad scripts for a short-form video creator.

Product:
[PRODUCT DETAILS]

Audience:
[AUDIENCE DETAILS]

Selected hook:
[HOOK]

Platform:
[INSTAGRAM / FACEBOOK / YOUTUBE SHORTS]

Write a 25-35 second UGC ad script using this structure:
1. Hook
2. Relatable problem
3. Product introduction
4. Benefit explanation
5. Visual demonstration
6. CTA

Include:
- Spoken script
- Visual direction
- On-screen text
- CTA

Tone:
Natural, friendly, honest, conversion-focused.
Avoid:
Medical claims, unrealistic promises, overhyped language.
```

## Prompt 4: Caption and CTA Generator

```text
Create platform-specific captions and CTAs for the UGC script below.

Script:
[PASTE SCRIPT]

Product:
[PRODUCT DETAILS]

Return:
1. Instagram caption
2. Facebook caption
3. YouTube Shorts caption
4. Five CTA options
5. Three hashtag sets

Keep captions clear, benefit-led, and suitable for paid or organic posting.
```
